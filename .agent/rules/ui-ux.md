# Reglas de UI/UX (EcoBot)

1. **Estética**: Moderna y minimalista. **EVITAR** diseños infantiles, apuntando a adolescentes (14-17 años).
2. **Esquema de Colores**: 
   - Modo oscuro para menús de pausa, inventario y HUD.
   - Utilizar fondos grises oscuros o negros con opacidad.
   - Contraste claro con tipografía blanca/clara para facilitar la lectura.
   - Aspecto profesional/tecnológico.
3. **HUD (Heads-Up Display)**:
   - Mostrar estado del inventario, porcentaje de contaminación global y puntos.
   - Renderizar en una `UIScene` paralela (separada de la cámara principal).
