# Reglas de Internacionalización (i18n)

1. **Idioma de Desarrollo**: Actualmente el proyecto está focalizado en español (para el público objetivo), pero los textos que se presenten en la UI del juego deben extraerse idealmente desde un objeto de configuración o archivo JSON, para facilitar una futura traducción.
2. **Textos del Juego**: Los textos de la interfaz (inventario, puntaje, mensajes de victoria/derrota) no deben estar hardcodeados directamente en los componentes visuales.
